# **App Name**: PropCall 360

## Core Features:

- Anti-Leakage Cloud Calling: Enable agents to make calls to clients without revealing the agent's actual phone number, preventing data theft.
- WhatsApp Automation Bot: Automatically send property brochures and welcome messages to new leads via WhatsApp upon lead generation.
- Offline 360° Virtual Tour: Allow agents to present pre-downloaded, high-quality 360° virtual tours to clients, even in areas with no network coverage.
- Real-Time Lead Dashboard: Provide agency owners with a real-time dashboard showing the number of leads generated, categorized by 'Hot' and 'Cold' status.
- Agent Tracking & Analytics: Track agent activities, including the number of calls made, call duration, and the number of virtual tours presented to clients.
- Follow-up Reminders: Send automated notifications to agents reminding them to follow up with specific clients based on their previous interactions.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and reliability in property dealings.
- Background color: Light gray (#F5F5F5) for a clean and professional look.
- Accent color: Warm orange (#FF9800) for call-to-action buttons and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its neutral and modern appearance.
- Use clear and professional icons for property features and agent actions.
- Design a clean and intuitive dashboard layout for easy navigation and quick access to key information.
- Subtle transitions and animations for a smooth and engaging user experience.
